# Folder media — imagini & video

Înlocuiește placeholderele din site cu fișiere reale. Pune fișierul cu **exact** numele de mai jos
în acest folder și el va apărea automat în site (nu trebuie să modifici codul).

Dacă un fișier lipsește, site-ul afișează un placeholder elegant cu numele fișierului așteptat.

## Imagini așteptate
| Fișier | Unde apare | Recomandare |
| --- | --- | --- |
| `hero.jpg` | Imaginea mare din partea de sus (hero) | Îngrijitoare cu un vârstnic, lumină caldă, orizontal, min. 1600px lățime |
| `despre.jpg` | Secțiunea „Cine suntem” | Echipa / birou / îngrijire la domiciliu |
| `mirela.jpg` | Portretul Mirelei Popescu | Portret vertical, fundal simplu |
| `beneficii.jpg` | Banda din secțiunea Beneficii | Casă/familie din Germania, cald |
| `germania.jpg` | Secțiunea „De ce să ne alegi” | Peisaj/oraș german sau document de muncă |

## Video (opțional)
| Fișier | Unde apare |
| --- | --- |
| `hero.mp4` | Dacă există, înlocuiește imaginea hero cu un video care rulează în fundal |
| `marturie-mirela.mp4` | Card video în mesajul Mirelei (mesaj video personal) |

## Note
- Formate acceptate: `.jpg`, `.png`, `.webp` pentru imagini; `.mp4`, `.webm` pentru video.
- Fiecare card de media afișează și calea fișierului (ex: `media/hero.jpg`) ca referință.
- Pentru cele mai bune rezultate, folosește fotografii reale cu îngrijitori și beneficiari (cu acordul lor).
